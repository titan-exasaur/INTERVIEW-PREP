# The Stack to Learn — Priority Order (Updated)

## Week 1–2 — GenAI basics (highest ROI right now)
- LLM fundamentals — tokens, context window, temperature, inference basics
- Embeddings + Vector DB — how retrieval works
- RAG pipeline — chunking, retrieval, response generation
- LangChain / LangGraph basics
- Prompt engineering basics
- Start building one project immediately, don’t wait for “full learning”



## Week 3–4 — Build One Real Project
- Pick one serious project and complete MVP
- Best option: Supply Chain / Forecasting Copilot (fits your background)
- Other options: Resume Q&A bot, Policy chatbot, SQL assistant
- Focus on backend logic, retrieval quality, usability



## Week 5 — Productionise the Project
- Docker — containerize app properly
- FastAPI / Flask API serving
- Environment variables / secrets handling
- Logging + error handling
- Basic testing



## Week 6 — CI/CD + Cloud
- GitHub Actions / basic CI/CD pipeline
- Auto build + test + deploy flow
- Pick ONE cloud: AWS or GCP
- Deploy as live endpoint / app



## Parallel Track — SQL + Interview Prep
- Practice SQL (joins, CTEs, window functions)
- Python coding basics for interviews
- ML system design stories from your real work
- Resume updates based on project
